## 前后端分离启动

+ 先在后端文件夹下按照文件夹中`readme.md`操作，启动后端服务器

+ 后端服务器的端口应为`http://localhost:5000/`

+ 在前端文件夹下先输入命令安装依赖

  + ```shell
    npm install 
    ```

+ 再启动前端

  + ```shell
    npm start
    ```

+ 之后访问前端服务器`http://localhost:3000/`
